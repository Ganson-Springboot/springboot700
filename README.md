## 4.1前台功能模块
就业信息管理系统，进入到系统首页，可以查看首页，招聘信息，交流论坛，公告资讯，个人中心，后台管理等内容进行操作，如图4-1所示。

图4-1系统首页界面图

![](/md/blog.012.png)

图4-2用户注册界面图

图4-3招聘信息界面图
